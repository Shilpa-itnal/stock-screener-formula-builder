# Stock Screener with Formula Builder

A full-stack stock screening and backtesting app where users can write Excel-like formulas such as:

PE < 20 AND ROE > 15

## Features
- Custom formula builder
- Stock screening
- Historical backtesting
- Performance charts
- Next.js frontend and API routes
- Web Worker based formula parsing
- PostgreSQL-ready structure

## Tech Stack
- Next.js
- TypeScript
- Web Workers
- Recharts
- PostgreSQL
- Financial Modeling Prep API

```bash
npm install
npm run dev
